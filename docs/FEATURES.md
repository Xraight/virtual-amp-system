# Manual de Funciones — Virtual Amp System

## Pestañas de Control

### 1. Amplificador (Main)
- **Gain**: Controla la cantidad de saturación de las etapas de pre-amp.
- **Tone Shaping**: Sección principal para esculpir el sonido (Bass, Mid, Treble, Presence).
- **Noise Gate**: Filtra el ruido de fondo. Ajusta el `Threshold` para silenciar el hum cuando no tocas.
- **Master**: Volumen final de la etapa de potencia.

### 2. Gabinete (Cab)
- **Microphone**: Selecciona entre SM57, U87, R121 y MD421. Cada uno varía la respuesta tonal.
- **Low/High Cut**: Elimina frecuencias extremas (sub-graves o agudos hirientes) para que la guitarra encaje en la mezcla.

### 3. Efectos (FX)
- **Overdrive**: Un pedal verde clásico para empujar el amplificador al límite.
- **Chorus**: Añade profundidad y modulación estéreo.
- **Delay**: Eco analógico con control de tiempo y repeticiones.

### 4. Ecualizador (EQ)
- Ecualizador gráfico de 6 bandas para el ajuste fino final del tono antes de la salida.

## Herramientas Adicionales

### El Tuner
Haz clic en el botón **TUNER** en la parte superior para activar la afinación. La pantalla mostrará la nota detectada y una barra de precisión. El audio pasará de forma limpia (bypass) mientras el afinador esté activo.

### Sistema de Presets
- **Menú Central**: Selecciona presets preestablecidos.
- **SAVE**: Guarda tus ajustes actuales en un archivo XML en tu computadora.
- **< y >**: Navega rápidamente entre los presets de la lista.

---
*Tip: Para sonidos Clean, mantén el Gain por debajo de 3. Para Metal, sube el Gain a 7 y reduce un poco los Mids.*
